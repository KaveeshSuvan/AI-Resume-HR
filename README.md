# Gear Up Resume 🚀

AI-powered Resume Analyzer & Career Roadmap Generator built with Flask, Python, HTML, CSS, and JavaScript.

## 🔍 Features
- 📄 Resume upload (PDF/Text)
- 🧠 Automatic skill extraction
- 📊 Resume vs Job Description matching
- ❌ Missing skill gap analysis
- 💼 Current & future job role suggestions
- 🗺️ Personalized learning roadmap with resource links
- ✍️ AI-generated improved ATS-friendly resume
- ⚡ Fast and responsive web interface

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- AI Integration: Local / Cloud LLM Models
- PDF Parsing: PyPDF2

## 🌍 SDG Goals
- SDG 4: Quality Education
- SDG 8: Decent Work & Economic Growth

## 🎯 Purpose
To help students, freshers, and professionals understand their resume strengths, identify skill gaps, and become job-ready with AI guidance.
